# アプリ概要
VRをつけて24時間疑似マラソンをしよう！
モバイルで位置情報を取得して、取得したコースを走れます。
座標を取得すれば好きな景色で走れます。
# アプリの遊び方
➀ログインする。
➁ホーム画面に遷移したらマラソンでスタートを押す
➂VRをつけて走ろう！
